# starlite-websockets-hello-world

Minimum Starlite WebSocket Implementation.

`$ poetry install`

`$ poetry run uvicorn main:app --reload`

Navigate to:

`localhost:8000/`

## Code Quality

After cloning:

`$ pre-commit install`

Run on all files:

`$ pre-commit run --all-files`

Run a specific hook:

`$ pre-commit run mypy --all-files`
